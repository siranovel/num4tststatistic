## install ##

From rubygems:  
~~~
    [sudo] gem install num4tststatistic
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4tststatistic.git  
    cd num4tststatistic  
    gem build *.gemspec
    gem install *.gem
~~~
